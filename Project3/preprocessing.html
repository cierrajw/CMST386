Preprocessing with SASS/LESS

What is preprocessing in CSS? Preprocessing allows you to generate CSS from the preprocessor's own unique syntax. For example with SASS, you have the ability to organize your CSS in many different ways that you can't with conventional CSS.

Ways to Implement Preprocessing

Ways to implement preprocessing:

  • Server:

    Is used when browsers have all code.

  • Client:

    Reduces client-side computation.

Mechanisms Used in SASS:

SASS allows for many different ways to organize your style/css. For example, you are able to create variables. One important principle in software engineering (and of course with CSS styling as well) is the DRY principle, or 'don't repeat yourself'. Variables come in handy when it comes to this principle. The variables allow for you to re-use certain style elements. SASS also allows for you to create functions, as well as to nest style elements as an alternative way to organize them. Nesting is one way to produce cleaner code.
